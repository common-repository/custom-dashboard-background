Custom dashboard background control for WordPress admin back-end backgrounds.

== Description ==

This plugin let's you set a background image that shows on every page inside of the WordPress admin back-end with automatic wrap styling.
See the screenshots for examples.

#### Features

- WordPress back-end pages background image
- Automatic content wrap styling
- Footer text color
- Content wrap border radius
- Black and white filter
- Blur filter

#### Development

This plugin is being developed on [GitHub](https://github.com/). 
Look at [boele/custom-dashboard-background](https://github.com/boele/custom-dashboard-background) if you want to collaborate.

== Installation ==

1. In your WordPress admin panel, go to *Plugins > New Plugin* and search for **Custom Dashboard Background** and click "*Install now*"
2. Or, download the plugin and upload the contents of `custom-dashboard-background.zip` to the folder `/wp-content/plugins/` of your WordPress website.
3. Go to *Plugins* and activate the **Custom Dashboard Background** plugin
4. Go to *Settings > Custom Dashboard Background* to configure this plugin.
